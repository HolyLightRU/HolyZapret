<h1 align="center">HolyZapret</h1>
Набор инструментов и экспериментов для обхода интернет-цензуры и ограничений, основанный на оригинальном проекте Zapret.

## ⚡️ Возможности

*   **Различные стратегии обхода:** Включает как портированные стратегии из оригинального репозитория, так и собственные разработки (смотрите папку `additional`).
*   **Возможность удобно развернуть локальный MTPROTO прокси для десктопной версии ТГ❕**
*   **Фикс звонков в Telegram через Zapret:** Работает в general.bat и auto_update.bat, требует наличие подобного фикса у второго собеседника.
*   **Восстановление работы Roblox:** Работает в auto_update.bat, MAX.bat и в additional выше 11.
*   **Сервис проверки доступности сайтов:** Удобный сервис (доступный в manager.bat) позволяет быстро определить и подобрать рабочую стратегию.
*   **Proxy Hosts System:** Система на основе файла `hosts` для доступа к заблокированным сайтам (например, chatgpt.com, open.spotify.com) и обхода ограничений TikTok без VPN. Список сервисов находится в `\lists\hosts-list.txt`.
*   **Service Manager:** Удобный менеджер для настройки и управления сервисом обхода, как в оригинальном проекте.
